# diary-app

A space to share your thoughts and memories anonymously, simply come up with a pseudonym and and start writing! 

# installation & usage 

to get started:
cd server 
docker compose up

to open the webpage:
cd client 
open index.html

when you are finished:
docker compose down --remove-orphans --volumes
